## Images

### Markdown Syntax

The simplest way to add an image:

```md
![alt text](/images/my-image.png)
```

Images must be under 5MB. For anything larger, host it on a service like [Cloudinary](https://cloudinary.com/) or [S3](https://aws.amazon.com/s3/) and use the URL.

### HTML Embeds

For more control over sizing and styling, use `<img>` tags directly:

```html
<img height="200" src="/images/my-image.png" alt="Description" />
```

### Light and Dark Mode

Display different images based on the user's color scheme:

```html
<img className="block dark:hidden" src="/images/hero-light.png" alt="Light mode hero" />
<img className="hidden dark:block" src="/images/hero-dark.png" alt="Dark mode hero" />
```

This is what I use on the homepage:

<img className="block dark:hidden" src="/images/hero-light.png" alt="humanlup docs hero — light" />
<img className="hidden dark:block" src="/images/hero-dark.png" alt="humanlup docs hero — dark" />

<Tip>
  Store all images in `/images/` and reference them with root-relative paths like `/images/filename.png`.
</Tip>

### The Frame Component

`<Frame>` adds a styled border and optional caption:

```html
<Frame caption="Deploy checks passing">
  <img src="/images/checks-passed.png" alt="Checks passed" />
</Frame>
```

<Frame caption="Deploy checks passing">
  <img src="/images/checks-passed.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

## Embeds

### iFrames

Embed external content like YouTube videos:

```html
<iframe
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video"
  width="560"
  height="315"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>
```

<Tip>
  Mintlify supports [HTML tags in Markdown](https://www.markdownguide.org/basic-syntax/#html), so you can embed anything that works in standard HTML.
</Tip>

## Best Practices

- **Alt text** — always include descriptive `alt` attributes for accessibility
- **File size** — keep images under 5MB; compress PNGs and use SVGs for logos
- **Format** — SVG for vectors, PNG for screenshots, JPG for photos
- **Dark mode** — provide light/dark variants for hero images and diagrams with high contrast
