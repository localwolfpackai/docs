You've probably heard people talk about "AI coding assistants" and wondered whether they're actually useful or just hype. I had the same question. Claude Code is the tool that changed my answer.

## The short version

Claude Code is an AI-powered coding tool made by [Anthropic](https://www.anthropic.com) that runs in your terminal. You describe what you want in plain English, and it reads your files, writes code, runs commands, and manages your project. It's not autocomplete. It's closer to having a collaborator who can see your whole codebase and actually do things with it.

<Info>
  You don't need to be a developer to understand this page. If you're learning to code, exploring AI tools, or just curious about how this stuff works, you're in the right place.
</Info>

## How it actually works

When you open Claude Code in your terminal, here's what happens behind the scenes:

<Steps>
  <Step title="You type what you want">
    Something like "add a dark mode toggle to the settings page" or "why is this test failing?"
  </Step>
  <Step title="Claude reads your codebase">
    It scans your files, understands the structure, and figures out what's relevant to your request.
  </Step>
  <Step title="It makes a plan">
    Before changing anything, Claude thinks through what needs to happen. You can even use Plan Mode to review the plan before it acts.
  </Step>
  <Step title="It takes action">
    Edits files, runs commands, creates new files. Each action asks for your permission first (unless you configure it otherwise).
  </Step>
  <Step title="You review and approve">
    Nothing ships without you saying so. You're always in control of what gets committed and pushed.
  </Step>
</Steps>

## Getting started

You'll need [Node.js](https://nodejs.org) (version 18 or higher) and either an Anthropic API key or a Claude subscription.

<CodeGroup>

```bash Install with npm
npm install -g @anthropic-ai/claude-code
```

```bash Install with the native installer
curl -fsSL https://claude.ai/install.sh | bash
```

```bash Install with Homebrew
brew install claude-code
```

</CodeGroup>

Once installed, navigate to any project folder and run:

```bash
claude
```

That's it. You're in. Type what you need and Claude takes it from there.

<Warning>
  **Don't use `sudo` to install.** If you get a permission error during install, use the native installer (the `curl` command above) instead. Running npm with `sudo` causes problems down the line.
</Warning>

## The CLAUDE.md file

This is one of those things that seems small but matters a lot. Every project can have a `CLAUDE.md` file at the root. It's like a briefing document that Claude reads at the start of every session. You put your preferences in there: how you like your code formatted, which tools to use, what the project structure looks like.

```markdown CLAUDE.md
# Project conventions
- Use TypeScript with strict mode
- Tests go in __tests__ directories
- Run `npm test` before committing

# Architecture
- State management: Zustand
- Styling: Tailwind CSS
- API calls go through src/api/
```

<Tip>
  Run `/init` inside Claude Code to generate a starter CLAUDE.md based on your current project. It gives you a solid starting point that you can refine over time.
</Tip>

## Is it free?

Claude Code requires either a [Claude Pro/Team subscription](https://claude.ai) or an [Anthropic API key](https://console.anthropic.com). With the API, you pay based on usage (tokens in and out). With a subscription, usage is included up to your plan limits.

The API route gives you more control over costs. The subscription route is simpler if you just want to use it without thinking about billing.
