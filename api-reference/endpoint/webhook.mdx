<Info>
  Webhooks are HTTP callbacks that notify your application when events occur. This webhook fires whenever a new task is created.
</Info>

## Usage Notes

- Your endpoint must return a `200` status code to acknowledge receipt. If the webhook doesn't receive a `200`, it may retry.
- The request body contains the full task object including `id`, `title`, `status`, and optional `priority`.
- Payloads are sent as `POST` requests with `Content-Type: application/json`.

### Example Payload

```json
{
  "id": 3,
  "title": "Deploy docs site update",
  "status": "pending",
  "priority": "high"
}
```

### Handling Webhooks

<CodeGroup>

```python Python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route("/webhook/task", methods=["POST"])
def handle_task_webhook():
    task = request.get_json()
    print(f"New task created: {task['title']}")
    return jsonify({"status": "received"}), 200
```

```javascript Node.js
const express = require("express");
const app = express();

app.use(express.json());

app.post("/webhook/task", (req, res) => {
  const task = req.body;
  console.log(`New task created: ${task.title}`);
  res.status(200).json({ status: "received" });
});
```
