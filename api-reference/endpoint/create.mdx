title: 'Create Task'
openapi: 'POST /tasks'
description: 'Create a new task'
icon: 'plus'
---

<Info>
  Send a JSON body with the task details. The `title` field is required. The server assigns a unique `id` and sets the initial status to `pending`.
</Info>

## Usage Notes

- The `title` field is required and must be a non-empty string.
- The `priority` field is optional — valid values are `low`, `medium`, or `high`.
- On success, the endpoint returns the created task including the server-assigned `id` and `status`.

### Example Request Body

```json
{
  "title": "Deploy docs site update",
  "priority": "high"
}
```

### Example Response

```json
{
  "id": 3,
  "title": "Deploy docs site update",
  "status": "pending",
  "priority": "high"
}
```
