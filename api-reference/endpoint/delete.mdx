---
title: 'Delete Task'
openapi: 'DELETE /tasks/{id}'
description: 'Remove a task by its ID'
icon: 'trash'
---

<Warning>
  This action is permanent. Once a task is deleted, it cannot be recovered. Make sure you have the correct task ID before calling this endpoint.
</Warning>

## Usage Notes

- The `id` path parameter is required and must be an integer.
- On success, the endpoint returns a `204 No Content` response with an empty body.
- If the specified `id` does not exist, the API returns a `400` error.

### Example Error Response

```json
{
  "error": 400,
  "message": "Task with the specified ID was not found"
}
```
